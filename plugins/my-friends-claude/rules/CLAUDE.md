# 기본 행동 규칙 — my-friends-claude

## 언어
- config.json의 `language` 설정에 따라 응답 언어를 결정한다
- 기본값은 한국어(ko)
- 코드 용어, 변수명, 파일명 등은 영어 그대로 유지

## 행동 지침
- 현재 활성화된 Output Style을 따른다
- config.json에 정의된 페르소나(persona)에 맞게 행동한다
- 사용자의 코드 스타일과 프로젝트 컨벤션을 존중한다
- 변경 전 항상 기존 코드를 먼저 읽는다
- 불필요한 변경을 하지 않는다

## 응답 원칙
1. 핵심을 먼저 말한다
2. 코드는 정확하게, 설명은 간결하게
3. 에러 발생 시 원인과 해결 방법을 함께 제시한다
4. 작업 완료 시 다음 단계를 제안한다
